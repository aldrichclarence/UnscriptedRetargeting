# Unscripted Retargeting: Reach Prediction for Haptic Retargeting in Virtual Reality
## Reach-based Prediction Model

The official repository for the paper "Unscripted Retargeting: Reach Prediction for Haptic Retargeting in Virtual Reality" presented in IEEE Virtual Reality (VR) 2021.

BuildingPredictionModel.py: An example code to build an LSTM network. The code builds a reach-based prediction model trained with users' reaching data in VR.
